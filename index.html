Что вы должны выучить

  • Как создать новый репозиторий

    • Чтобы создать git репозиторий из каталога, выполните команду git init.

  • Как создать удаленный репозиторий

    • Чтобы создать удаленный репозиторий перейдите на https://github.com и войдите в свой аккаунт. Нажмите кнопку New repository (Новый репозиторий). На открывшейся странице введите имя репозитория (Repository name) и нажмите кнопку Create repository.
