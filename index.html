Что вы должны выучить

  • Как создать новый репозиторий

    • Чтобы создать git репозиторий из каталога, выполните команду git init.

  • Как создать удаленный репозиторий

    • Чтобы создать удаленный репозиторий перейдите на https://github.com и войдите в свой аккаунт. Нажмите кнопку New repository (Новый репозиторий). На открывшейся странице введите имя репозитория (Repository name) и нажмите кнопку Create repository.

  • Как сопоставить два репозитория между собой

    • При создании удаленного репозитория, нужно выполнить следующие команды в терминале:

    1. git remote add origin https://github.com/.../....git (https://github.com/Ann-Fox/Course-3-MODULE-4-GIT-Team-Development.git)
    2. git branch -M main
    3. git push -u origin main
  • Какой командой можно исправить последний коммит, который вы еще не отправили в удаленный репозиторий

    • git commit --amend -m 'Новый комментарий'

  • Какой командой получить все изменения из удаленного репозитория, какой командой применить эти изменения

    • git log

  • Какой командой создать новую ветку

    • git branch name-new-branch

  • Какой командой переключиться на другую ветку или коммит

    • git checkout name-branch or num-commit
